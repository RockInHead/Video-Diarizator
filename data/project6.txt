Проект: Система бронирования отелей  
Необходимо разработать систему, позволяющую пользователям удобно бронировать номера в отелях, с учетом различных параметров и доступности. Используемый стек: ASP.NET для создания надежного бэкенда и React Native для разработки мобильного приложения. Важно подключить API для получения данных о доступности номеров, что обеспечит актуальную информацию для пользователей. Я представляю компанию "HotelBooking".
