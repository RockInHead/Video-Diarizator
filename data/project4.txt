Проект: Приложение для учета расходов  
Разработка приложения, которое поможет пользователям отслеживать свои расходы и доходы с возможностью составления отчетов и анализа финансового состояния. Используемые технологии: Node.js для серверной части, Express для создания RESTful API и MySQL для хранения данных. В приложении реализованы графики для визуализации данных, что делает анализ финансов более наглядным и доступным. Я — фрилансер, работающий над проектом.
